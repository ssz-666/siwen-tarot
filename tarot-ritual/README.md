# 思文塔罗

iPhone-first local tarot app resources for the `SiwenTarot` Swift shell.

## Included

- Ritual-style black and gold UI for asking a question, choosing a spread, drawing cards, reading, and follow-up questions.
- 78-card Rider-Waite library using local images under `assets/rider-waite/`.
- Local fallback reading plus optional DeepSeek API integration through the native WebView bridge.
- Daily card, card library, and local API settings views.

The iOS app loads `index.html` directly from this folder. No build step is required for the static app.
